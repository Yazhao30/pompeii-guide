# 庞贝随行导览（PWA）

这是一个无需安装应用商店客户端的手机网页小程序，包含：

- OpenStreetMap 地图与 25 个庞贝主要地点
- 浏览器实时定位、位置精度圈、最近地点和方向距离
- 中文建筑讲解与“现场重点看什么”
- 2小时、3小时、5小时推荐路线
- 收藏、到访记录和本地保存
- PWA 安装与应用外壳离线缓存

## 本机运行

定位功能通常要求 HTTPS 或 localhost，不能直接双击 `index.html`。

```bash
cd pompeii-guide
python -m http.server 8000
```

然后在浏览器打开：

```text
http://localhost:8000
```

## 部署到手机可访问的网址

可把整个文件夹上传到 GitHub Pages、Cloudflare Pages、Netlify 或任意 HTTPS 静态网站。部署后，用手机浏览器打开并选择“添加到主屏幕”。

## 重要说明

- 坐标用于现场辅助定向，古城内 GPS 可能有 5–25 米误差。
- 路线连线表示参观顺序，不代表可以穿越建筑或封闭道路。
- 开放状态会随修复、活动和现场管理变化，出发前应查看庞贝考古公园官方通知。
- 本应用为个人旅行辅助工具，不是庞贝考古公园官方产品。
